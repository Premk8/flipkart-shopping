# 🛒 Flipkart Shopping App

A simple **Flipkart-style shopping web application** built using **Python and Streamlit.  
This project simulates an online shopping experience where users can view products, add them to a cart, and generate a bill with discount and GST calculation.

---

## 🚀 Features

- 📦 Display available products with price and stock
- 🛍 Add products to cart with selected quantity
- ❌ Remove items from cart
- 🧾 Automatic invoice generation
- 💰 Billing system with:
  - **10% Discount** if subtotal exceeds ₹50,000
  - **18% GST calculation**
- 🗑 Clear cart option
- 🔄 Automatic stock update when items are added or removed
